Purpose:
Generate a flattened device tree (FDT/DTB) which gives the Linux Kernel the information it needs to utilize your devices hardware.

The Easy Way:
Download the DTB file corresponding to your device from the device_trees folder.

The Hard Way:
1. Download the arm-linux-gnueabihf cross chain for your platform from linaro from https://releases.linaro.org/components/toolchain/binaries/latest/arm-linux-gnueabihf/
2. Unpack the the archive.
3. Locate the source files for the desired kernel version, the kernel-image package page has links to the corresponding source ex. https://packages.debian.org/stretch/linux-image-4.9.0-8-armmp
4. Download the source files (*.orig.tar.xz, *.debian.tar.xz, *.dsc)
5. Unpack the kernel source:
   dpkg-source -x linux_x.xx.xx-x.dsc 
6. Enter the linux_x.xx.xx-x directory.
7. Copy the armmp config file to .config
   cp debian/config/armhf/config .config
8. run "make oldconfig"
